# @layerone-theme/ui-angular

Angular component library for the LayerOne design system.

## Install
```bash
npm install @layerone-theme/ui-angular
```

## Usage
Import the styles once in your app (global styles):
```css
@import "@layerone-theme/theme/dist/index.css";
```

Then import components/modules from this package:
```ts
import { LayerOneButtonModule } from "@layerone-theme/ui-angular";
```
